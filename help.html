How to Use the Extension

Secondary-click on an image from a webpage and select "Predict Image Authenticity" from the Context Menu. A pop-up with the "Manipulated Image Detector" will open, the image will automatically be given to the model, and the model will predict the authenticity of the image.

Click here to open the "Manipulated Image Detector" as a new tab of your browser.

How Does the Detector Work?

This Chrome extension is meant to be a companion to the "Manipulated Image Detector and Recommender" Android application.

It uses an on-device model that was built to detect whether an image has been manipulated or edited in any way. The model is a custom convolutional neural network that was trained on 3938 authentic images and 3938 manipulated images from the CASIA2 dataset. The model, once fully trained, achieved an F1 score of ~87.3% when tested on another subset of images from the CASIA2 dataset it was not explicitly trained on.

Remember that all detections made by the model are, in fact, predictions.
